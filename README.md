# shop-easy
The platform includes essential shopping features such as product listing, product detail view, add-to-cart functionality, and a user-friendly interface.

This project simulates a basic online shopping platform where users can:

Browse products

View product details

Add and remove items from the cart

See total price calculation

Navigate through a clean and responsive layout
